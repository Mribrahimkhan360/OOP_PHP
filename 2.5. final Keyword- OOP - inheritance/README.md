<p>কোনো মেথডকে <strong>final</strong> ঘোষণা করলে, সেই মেথডটি subclass-এ ওভাররাইড করা যাবে না। অর্থাৎ, parent class থেকে প্রাপ্ত সেই মেথডটি পরিবর্তন করা সম্ভব নয় এবং এটি মূল মেথডের মতোই কাজ করবে।</p>
